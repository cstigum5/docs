## List Account Logs

Get a list of account logs for an account.

### Query Parameters
<ParamField query="queryType" type="string">
Query type.
</ParamField>

<ParamField query="startTime" type="string" required>
Timestamp in UTC.
</ParamField>

<ParamField query="endTime" type="string" required>
Timestamp in UTC.
</ParamField>
